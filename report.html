Pré-processar Colunas de Atributo¶

Como voc pode ver, h muitas colunas no numricas que precisam ser convertidas! Muitas delas so simplesmente yes/no, por exemplo, a coluna internet. razovel converter essas variveis em valores (binrios) 1/0.

Outras colunas, como Mjob e Fjob, tm mais do que dois valores e so conhecidas como variveis categricas. A maneira recomendada de lidar com esse tipo de coluna criar uma quantidade de colunas proporcional aos possveis valores (por exemplo, Fjob_teacher, Fjob_other, Fjob_services, etc), e assinalar 1 para um deles e 0 para todos os outros.

Essas colunas geradas so por vezes chamadas de variveis postias (dummy variables), e ns iremos utilizar a funo pandas.get_dummies() para fazer essa converso. Execute a clula de cdigo abaixo para executar a rotina de pr-processamento discutida nesta seo.

def preprocess_features(X):
    ''' Pr-processa os dados dos estudantes e converte as variveis binrias no numricas em
        variveis binrias (0/1). Converte variveis categricas em variveis postias. '''
    
    # Inicialize nova sada DataFrame
    output = pd.DataFrame(index = X.index)

    # Observe os dados em cada coluna de atributos 
    for col, col_data in X.iteritems():
        
        # Se o tipo de dado for no numrico, substitua todos os valores yes/no por 1/0
        if col_data.dtype == object:
            col_data = col_data.replace(['yes', 'no'], [1, 0])

        # Se o tipo de dado for categrico, converta-o para uma varivel dummy
        if col_data.dtype == object:
            # Example: 'school' => 'school_GP' and 'school_MS'
            col_data = pd.get_dummies(col_data, prefix = col)  
        
        # Rena as colunas revisadas
        output = output.join(col_data)
    
    return output

def train_classifier(clf, X_train, y_train):
    ''' Ajusta um classificador para os dados de treinamento. '''
    
    # Inicia o relgio, treina o classificador e, ento, para o relgio
    start = time()
    clf.fit(X_train, y_train)
    end = time()
    
    # Imprime os resultados
    print "O modelo foi treinado em {:.4f} segundos".format(end - start)

    
def predict_labels(clf, features, target):
    ''' Faz uma estimativa utilizando um classificador ajustado baseado na pontuao F1. '''
    
    # Inicia o relgio, faz estimativas e, ento, o relgio para
    start = time()
    y_pred = clf.predict(features)
    end = time()
    
    # Imprime os resultados de retorno
    print "As previses foram feitas em {:.4f} segundos.".format(end - start)
    return f1_score(target.values, y_pred, pos_label='yes')


def train_predict(clf, X_train, y_train, X_test, y_test):
    ''' Treina e faz estimativas utilizando um classificador baseado na pontuao do F1. '''
    
    # Indica o tamanho do classificador e do conjunto de treinamento
    print "Treinando um {} com {} pontos de treinamento. . .".format(clf.__class__.__name__, len(X_train))
    
    # Treina o classificador
    train_classifier(clf, X_train, y_train)
    
    # Imprime os resultados das estimativas de ambos treinamento e teste
    print "Pontuao F1 para o conjunto de treino: {:.4f}.".format(predict_labels(clf, X_train, y_train))
    print "Pontuao F1 para o conjunto de teste: {:.4f}.".format(predict_labels(clf, X_test, y_test))
